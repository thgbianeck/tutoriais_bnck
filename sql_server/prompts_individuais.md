## FinanceDB — Prompts Individuais por Aula
## SQL Server para Aplicações Financeiras com T-SQL

---

## Como Usar Este Arquivo

Este arquivo contém os prompts individuais para geração de cada aula do curso. Para usar, copie o prompt da aula desejada e envie ao tutor, anexando sempre o arquivo **plano_mestre.txt** e o **log_estado_projeto.md** atualizado para garantir continuidade e contexto. Aguarde a confirmação do tutor antes de iniciar cada aula.
